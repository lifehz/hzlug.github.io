Package Control

Package Control是帮Sublime管理插件的那个插件。

命令行安装

打开sublime text ，按ctrl+(Mac为control + )或者菜单View > Show Console打开命令窗口，粘贴以上代码并回车即可。

或者

手动安装

1、点击 Preferences > Browse Packages… 菜单

2、进入 Installed Packages/ 目录

3、下载 Package Control.sublime-package，并复制文件到 Installed Packages/ 目录

4、重启 Sublime Text

安装插件

Sublime有两种安装插件的方式：

利用Package Control安装（推荐）

1、在Sublime中，打开插件管理器（command+shift+p）
2、输入Install Package
3、输入你想要的插件名称进行检索，选中想要的项按回4、车直接进行安装。
5、安装完成后，重新启动Sublime

手动安装

1、下载插件的源文件，将其拷贝到packages目录下。

2、完成后，重新启动Sublime
