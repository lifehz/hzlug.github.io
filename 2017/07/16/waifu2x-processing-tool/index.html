2017-07-16


waifu2x -在线放大图片，最大支持到两倍网络上找的图片不够大，直接拉又会很糊？
可以试试看 waifu2x 这个线上解析图片的服务，透过这个网站的计算方式，可以让图片在尽量不要模糊、像素化的情况下，让图片最高放大到两倍。对于做简报、报告需要图片素材的朋友应该会有帮助。
waifu2x 在向量类型（线条、色块组合）的图片效果最好，如果是要放大照片，则会有一点向量化的感觉。但是一般使用应该是没有问题的。
